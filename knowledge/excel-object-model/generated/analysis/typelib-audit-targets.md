# Type-library audit targets

Generated by `excel-com-kb analyze`; do not edit by hand. Targets define evidence to collect later; they assert no installed type-library facts.

Evidence labels are deliberate: **Documentation-established** is extracted from the pinned VBA documentation; **Project classification** is a planning label; **Type-library verification required** and **Runtime verification required** are open gates. No production API, COM activation, or type-library inspection is performed.

| Priority | Target set | Required audit facts | Why |
| --- | --- | --- | --- |
| 1 | `Excel.Application` and root candidate members | Type-library verification required: coclass/CLSID, interface IID, DISPIDs, invoke kinds, property/method collisions. | Dynamic root calls cannot be designed from documentation alone. |
| 2 | candidate 0.1 member records | Type-library verification required: parameter type, direction, optional/default encoding, return type, aliases. | 50 current canonical candidate records require confirmation before typed wrappers. |
| 3 | `Workbooks`, `Worksheets`, `Sheets`, `Windows`, `Names`, `ListObjects`, `ChartObjects`, `Shapes` | Type-library verification required: default member, item return interface, `_NewEnum`, count type, collection interface. | Collection semantics are deliberately unclaimed. |
| 4 | `Range.Value`, `Value2`, `Formula`, `Formula2`, `Find`, `Sort`, and export methods | Type-library verification required: VARIANT/SAFEARRAY signatures, optional argument representation, return type. | Range transport and optional invocation need a separate evidence base. |
| 5 | application/workbook/worksheet event interfaces | Type-library verification required: source IID, DISPIDs, connection points, callback signatures. | Events are deferred and cannot be inferred from page names. |
| 6 | adjacent chart, name, shape, table, pivot, query, and connection families | Type-library verification required: interface identity, aliases, member signatures. | Deferred breadth; no wrapper generation is authorized. |
